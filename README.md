# Nao

Parser for NAO player logs.

## Installation

Add this line to your application's Gemfile:

    gem 'nao'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install nao

## Usage

```ruby
  require 'nao'

  a = Nao.parse("mgv")
```
